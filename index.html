Version History

  • Version 1.1.1 — To be announced
    • Added "Jetbridges" so passengers can easily board their flight... wait, why are they like that?
  • Version 1.1.0 — December 4, 2025
    • Added "Element Properties" button to the header for easier navigation
    • Added 2 new elements: Hangar, Reference Image
    • Added the "Element Clipboard" section where you can copy and paste elements
    • Fixed the spawning of the sample airport to only spawn when there is no airport in the build area.
    • Fixed the saving and importing so that it also saves an apron's maximum aircraft handling type
  • Version 1.0.3 — November 21, 2025
    • Added the feature to save and import airports as JSON files
    • Changed gradient colors of elements to solid colors
  • Version 1.0.2 — November 11, 2025
    • Added an "Add Element" section and removed the original add elements from the ribbon and replaced them with "+Element"
    • Elements now spawn at the center of the display, so that you now don't have to look around for a new element
  • Version 1.0.1 — October 25, 2025
    • Fixed runway numbering and parallel runway numbering
    • Changed taxiway block color to yellow
    • Added apron/stand numberings
    • Reduced minimum width of an element to 30m (except for terminal buildings)
  • Version 1.0.0 — October 5, 2025
    • Initial release of Airport Builder with core features
